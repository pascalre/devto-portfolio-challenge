About

I work at the intersection of platform engineering, messaging infrastructure, and developer enablement. My background includes operating large-scale event-driven systems in production, with a strong focus on reliability, security, and observability.

This portfolio is containerized and deployed on Google Cloud Run. It features an integrated AI Agent Mesh to demonstrate modern cloud-native patterns.

When a user submits a query, the Go-based backend performs a concurrent fan-out: it immediately calls the Google Gemini API to ensure a low-latency synchronous response for the user, while simultaneously publishing the event to Google Cloud Pub/Sub. This decoupling demonstrates how modern enterprise architectures trigger background processes - like analytics, logging, or secondary agent tasks - without blocking the critical path of the user experience.
